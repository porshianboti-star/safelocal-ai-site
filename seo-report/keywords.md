# Tracked search keywords

These are the keywords the daily SEO report tracks (from
`SafeLocal-Project-Handoff.md` §5). **To add/remove a keyword, edit the list
below** — each item must be a markdown list item wrapped in backticks
(`` - `keyword` ``). The automation parses exactly these lines.

## High-value intent

- `on-premise llm for enterprise`
- `private ai automation infrastructure`
- `self-hosted local ai for business`
- `secure offline ai assistant`
- `local llm data privacy compliance`

## High-volume / alternative traffic

- `offline ai assistant`
- `local ai for windows`
- `private ai locally`
- `ai without internet`
- `chatgpt offline alternative`
- `run llm locally`
- `local llm gui`
